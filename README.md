# ✈️ Dream Travel Planner

Dream Travel Planner is a responsive frontend web application that helps users explore travel destinations, plan trips, and manage travel budgets in one place.

## 🚀 Features

* Explore popular travel destinations
* Plan trips with dates and duration
* Track travel expenses and budget
* View destination images in a gallery
* Read and submit traveler reviews
* Fully responsive design for mobile, tablet, and desktop

## 🛠 Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (ES6)
* LocalStorage API

## ▶️ How to Run

1. Clone the repository

git clone https://github.com/DishaAgarwalla/Dream-Travel-Planner.git

2. Open the project folder

3. Run **index.html** in your browser.
